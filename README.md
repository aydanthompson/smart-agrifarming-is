# Smart Agrifarming - Cattle Activity Analysis

Developed as part of the final assignment for the Information Systems (IS) module at the University of Suffolk.

## Data Sources

The cow IMU dataset was sourced from "[A dataset for detecting walking, grazing, and resting behaviors in free-grazing cattle using IoT collar IMU signals](https://www.frontiersin.org/journals/veterinary-science/articles/10.3389/fvets.2025.1630083/full)", the dataset from the paper is stored on [GitHub](https://github.com/WASP-lab/db-cow-walking).
